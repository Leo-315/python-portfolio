# Project: AI Test Score Prediction Model

## Problem
The goal of this project is to see if a computer can predict a student's test score
based on how many hours they studied and how much sleep they got. 
## Solution
I created a dataset using Google Sheets that includes hours studied, hours slept,
and test scores for multiple students. I then used Python and a learning model from scikit-learn to train the AI on this data. After training,
the model can predict a test score for a new student based on their study and sleep time.
## Tools Used
- Python
- Thonny
- Pandas library
- Scikit-learn library
- Google Sheets (for creating the dataset)
